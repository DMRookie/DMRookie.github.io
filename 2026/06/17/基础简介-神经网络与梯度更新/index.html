大模型、深度学习、人工智能……这些词我们经常听到，但它们背后的核心是什么？答案是：神经网络。神经网络是现代 AI 的基础。从手机上的语音助手，到 AlphaGo 战胜围棋世界冠军，再到 ChatGPT 能写诗、编程、回答问题，背后都是神经网络在"思考"。那么，神经网络到底是什么？它是如何"学习"的？这篇文章将用一个生活中的例子——预测孩子身高，带你从零开始理解神经网络的核心概念：

  • 神经网络：一个由简单计算单元组成的"大脑"
  • 梯度：告诉模型"该往哪个方向调整"的信号
  • 反向传播：高效计算梯度的巧妙算法

不需要任何数学基础，让我们开始吧。

1. 从身高预测开始

假设你想预测你未来孩子的身高。一个简单的想法是：孩子的身高和父母身高有关。于是你写出一个公式：

孩子身高=w1×父亲身高+w2×母亲身高\text{孩子身高} = w_1 \times \text{父亲身高} + w_2 \times \text{母亲身高} 孩子身高=w1​×父亲身高+w2​×母亲身高

这个公式就是一个线性函数。比如设 w1=0.5w_1 = 0.5w1​=0.5，w2=0.5w_2 = 0.5w2​=0.5，那么父亲178cm、母亲165cm的组合，预测孩子身高为 0.5×178+0.5×165=171.50.5 \times 178 + 0.5 \times 165 = 171.50.5×178+0.5×165=171.5cm。

这个简单的公式就是一个模型，具体来说，这是一个单层神经网络。父亲身高、母亲身高是输入，孩子身高是输出，w1w_1w1​、w2w_2w2​ 是模型的参数（也叫权重）。

2. 线性不够，需要激活函数

但你很快发现，实际情况没这么简单。父母身高高，孩子大概率高，但不是简单的线性倍数关系。身高的预测存在一个非线性的关系，使得下一代的身高回归平均水平，即会一个"天花板"效应——父母都超高时，孩子不一定比父母更高。

于是我们需要引入激活函数，把线性计算的结果做一下非线性变换：

预测身高=f(w1×父亲身高+w2×母亲身高)\text{预测身高} = f(w_1 \times \text{父亲身高} + w_2 \times \text{母亲身高}) 预测身高=f(w1​×父亲身高+w2​×母亲身高)

激活函数 fff 可以是 ReLU（f(x)=max⁡(0,x)f(x) = \max(0, x)f(x)=max(0,x)）、Sigmoid（f(x)=11+e−xf(x) = \frac{1}{1+e^{-x}}f(x)=1+e−x1​）等。它的作用是让模型能拟合更复杂的关系。

3. 更多因素：多层网络

现实中的身高预测还要考虑更多因素。比如孩子的身高还和祖父母的身高有关。于是模型变得更复杂：

第一层：从祖父母身高预测父母身高

父亲身高=f(w11×爷爷身高+w12×奶奶身高)\text{父亲身高} = f(w_{11} \times \text{爷爷身高} + w_{12} \times \text{奶奶身高}) 父亲身高=f(w11​×爷爷身高+w12​×奶奶身高)

母亲身高=f(w13×外公身高+w14×外婆身高)\text{母亲身高} = f(w_{13} \times \text{外公身高} + w_{14} \times \text{外婆身高}) 母亲身高=f(w13​×外公身高+w14​×外婆身高)

第二层：从父母身高预测孩子身高

孩子身高=f(w21×父亲身高+w22×母亲身高)\text{孩子身高} = f(w_{21} \times \text{父亲身高} + w_{22} \times \text{母亲身高}) 孩子身高=f(w21​×父亲身高+w22​×母亲身高)

这就是一个两层神经网络。每一层都有自己的参数，一层的输出作为下一层的输入。

更一般地，一个 LLL 层的神经网络可以写成：

h1=f(W1x)h_1 = f(W_1 x) h1​=f(W1​x)

h2=f(W2h1)h_2 = f(W_2 h_1) h2​=f(W2​h1​)

⋮\vdots ⋮

y^=f(WLhL−1)\hat{y} = f(W_L h_{L-1}) y^​=f(WL​hL−1​)

其中 xxx 是输入，y^\hat{y}y^​ 是输出，W1,W2,…,WLW_1, W_2, \ldots, W_LW1​,W2​,…,WL​ 是每一层的参数矩阵。

4. 如何调整参数：梯度下降

上面的网络有很多参数（w1,w2,W1,W2,…w_1, w_2, W_1, W_2, \ldotsw1​,w2​,W1​,W2​,…）。这些参数怎么确定呢？

核心思想：用已有数据来训练。比如你收集了1000个家庭的祖父母、父母、孩子身高数据。对每个家庭，计算预测值和真实值的差距（损失函数）：

L=12(y^−y)2\mathcal{L} = \frac{1}{2}(\hat{y} - y)^2 L=21​(y^​−y)2

然后想办法调整参数，让损失尽量小。

关键问题：怎么调整每个参数？

答案是：沿着损失下降最快的方向调。具体来说，对每个参数 wiw_iwi​，计算损失函数对它的偏导数 ∂L∂wi\frac{\partial \mathcal{L}}{\partial w_i}∂wi​∂L​，然后：

wi←wi−η⋅∂L∂wiw_i \leftarrow w_i - \eta \cdot \frac{\partial \mathcal{L}}{\partial w_i} wi​←wi​−η⋅∂wi​∂L​

其中 η\etaη 是学习率（步长大小）。这个过程叫梯度下降。

5. 梯度是什么

梯度就是损失函数对每个参数的偏导数。它告诉你：如果稍微增加一点参数 wiw_iwi​，损失会怎么变化。

举个直观的例子：你在山上，想最快下山。你看看四周哪个方向最陡（导数最大），就往那个方向走一步。梯度就是告诉你"哪个方向最陡"的那个量。

对于身高预测的例子：

  • 如果 ∂L∂w1=5\frac{\partial \mathcal{L}}{\partial w_1} = 5∂w1​∂L​=5，意思是 w1w_1w1​ 太大会让预测误差变大，需要减小w1w_1w1​ 的值
  • 如果 ∂L∂w1=−3\frac{\partial \mathcal{L}}{\partial w_1} = -3∂w1​∂L​=−3，意思是 w1w_1w1​ 太小会让预测误差变大，需要增大w1w_1w1​ 的值
  • 根据梯度正负，决定 w1w_1w1​ 该增大还是减小

6. 反向传播：如何高效计算梯度

在多层网络中，参数可能有成千上万个。如果对每个参数都单独算导数，计算量会非常大。

反向传播利用链式法则，从输出层开始，逐层往回算梯度。这是深度学习高效训练的关键。

6.1 用身高例子说明

假设我们有：

a=父亲身高=178a = \text{父亲身高} = 178 a=父亲身高=178

b=母亲身高=165b = \text{母亲身高} = 165 b=母亲身高=165

c=w1a+w2bc = w_1 a + w_2 b c=w1​a+w2​b

y^=f(c)\hat{y} = f(c) y^​=f(c)

L=12(y^−y)2\mathcal{L} = \frac{1}{2}(\hat{y} - y)^2 L=21​(y^​−y)2

设 w1=0.5w_1 = 0.5w1​=0.5，w2=0.5w_2 = 0.5w2​=0.5，真实孩子身高 y=175y = 175y=175，激活函数 f(x)=xf(x) = xf(x)=x（方便讲解，使用线性函数）。

前向传播：

  • c=0.5×178+0.5×165=171.5c = 0.5 \times 178 + 0.5 \times 165 = 171.5c=0.5×178+0.5×165=171.5
  • y^=171.5\hat{y} = 171.5y^​=171.5
  • L=12(171.5−175)2=7.225\mathcal{L} = \frac{1}{2}(171.5 - 175)^2 = 7.225L=21​(171.5−175)2=7.225

反向传播（从后往前算）：

  1. 损失对预测值的梯度：

    ∂L∂y^=y^−y=171.5−175=−3.5\frac{\partial \mathcal{L}}{\partial \hat{y}} = \hat{y} - y = 171.5 - 175 = -3.5 ∂y^​∂L​=y^​−y=171.5−175=−3.5

  2. 预测值对 ccc 的梯度（假设 fff 是恒等函数，导数为1）：

    ∂y^∂c=1\frac{\partial \hat{y}}{\partial c} = 1 ∂c∂y^​​=1

  3. 由链式法则，损失对 ccc 的梯度：

    ∂L∂c=∂L∂y^⋅∂y^∂c=−3.5×1=−3.5\frac{\partial \mathcal{L}}{\partial c} = \frac{\partial \mathcal{L}}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial c} = -3.5 \times 1 = -3.5 ∂c∂L​=∂y^​∂L​⋅∂c∂y^​​=−3.5×1=−3.5

  4. ccc 对 w1w_1w1​ 的梯度：

    ∂c∂w1=a=178\frac{\partial c}{\partial w_1} = a = 178 ∂w1​∂c​=a=178

  5. 由链式法则，损失对 w1w_1w1​ 的梯度：

    ∂L∂w1=∂L∂c⋅∂c∂w1=−3.5×178=−623\frac{\partial \mathcal{L}}{\partial w_1} = \frac{\partial \mathcal{L}}{\partial c} \cdot \frac{\partial c}{\partial w_1} = -3.5 \times 178 = -623 ∂w1​∂L​=∂c∂L​⋅∂w1​∂c​=−3.5×178=−623

  6. 类似地，损失对 w2w_2w2​ 的梯度：

    ∂L∂w2=−3.5×165=−577.5\frac{\partial \mathcal{L}}{\partial w_2} = -3.5 \times 165 = -577.5 ∂w2​∂L​=−3.5×165=−577.5

更新参数（设学习率 η=0.001\eta = 0.001η=0.001）：

  • w1←0.5−0.001×(−623)=0.5623w_1 \leftarrow 0.5 - 0.001 \times (-623) = 0.5623w1​←0.5−0.001×(−623)=0.5623
  • w2←0.5−0.001×(−577.5)=0.5578w_2 \leftarrow 0.5 - 0.001 \times (-577.5) = 0.5578w2​←0.5−0.001×(−577.5)=0.5578

可以看到，w1w_1w1​ 和 w2w_2w2​ 都变大了，这意味着模型认为应该给父母身高更大的权重，以提高预测值。

7. 各种运算的梯度

在实际神经网络中，会用到加、减、乘、除、激活函数等各种运算。下面逐一介绍它们的梯度计算和传播方式。

7.1 加法 c=a+bc = a + bc=a+b

前向：c=a+bc = a + bc=a+b

反向：

  • ∂c∂a=1\frac{\partial c}{\partial a} = 1∂a∂c​=1
  • ∂c∂b=1\frac{\partial c}{\partial b} = 1∂b∂c​=1

梯度直接传递给两个输入：

  • ∂L∂a=∂L∂c×1=∂L∂c\frac{\partial \mathcal{L}}{\partial a} = \frac{\partial \mathcal{L}}{\partial c} \times 1 = \frac{\partial \mathcal{L}}{\partial c}∂a∂L​=∂c∂L​×1=∂c∂L​
  • ∂L∂b=∂L∂c×1=∂L∂c\frac{\partial \mathcal{L}}{\partial b} = \frac{\partial \mathcal{L}}{\partial c} \times 1 = \frac{\partial \mathcal{L}}{\partial c}∂b∂L​=∂c∂L​×1=∂c∂L​

特点：梯度平均分配到两个输入。

7.2 乘法 c=a×bc = a \times bc=a×b

前向：c=a×bc = a \times bc=a×b

反向：

  • ∂c∂a=b\frac{\partial c}{\partial a} = b∂a∂c​=b
  • ∂c∂b=a\frac{\partial c}{\partial b} = a∂b∂c​=a

梯度传递：

  • ∂L∂a=∂L∂c×b\frac{\partial \mathcal{L}}{\partial a} = \frac{\partial \mathcal{L}}{\partial c} \times b∂a∂L​=∂c∂L​×b
  • ∂L∂b=∂L∂c×a\frac{\partial \mathcal{L}}{\partial b} = \frac{\partial \mathcal{L}}{\partial c} \times a∂b∂L​=∂c∂L​×a

特点：乘法的梯度传递依赖另一个输入的值。

7.3 幂运算 c=abc = a^bc=ab

前向：c=abc = a^bc=ab

反向：

  • ∂c∂a=b⋅ab−1\frac{\partial c}{\partial a} = b \cdot a^{b-1}∂a∂c​=b⋅ab−1
  • ∂c∂b=ab⋅ln⁡(a)\frac{\partial c}{\partial b} = a^b \cdot \ln(a)∂b∂c​=ab⋅ln(a)

梯度传递：

  • ∂L∂a=∂L∂c×b⋅ab−1\frac{\partial \mathcal{L}}{\partial a} = \frac{\partial \mathcal{L}}{\partial c} \times b \cdot a^{b-1}∂a∂L​=∂c∂L​×b⋅ab−1
  • ∂L∂b=∂L∂c×ab⋅ln⁡(a)\frac{\partial \mathcal{L}}{\partial b} = \frac{\partial \mathcal{L}}{\partial c} \times a^b \cdot \ln(a)∂b∂L​=∂c∂L​×ab⋅ln(a)

常用特例：c=a2c = a^2c=a2，∂L∂a=∂L∂c×2a\frac{\partial \mathcal{L}}{\partial a} = \frac{\partial \mathcal{L}}{\partial c} \times 2a∂a∂L​=∂c∂L​×2a

7.4 除法 c=abc = \frac{a}{b}c=ba​

前向：c=abc = \frac{a}{b}c=ba​

反向：

  • ∂c∂a=1b\frac{\partial c}{\partial a} = \frac{1}{b}∂a∂c​=b1​
  • ∂c∂b=−ab2\frac{\partial c}{\partial b} = -\frac{a}{b^2}∂b∂c​=−b2a​

梯度传递：

  • ∂L∂a=∂L∂c×1b\frac{\partial \mathcal{L}}{\partial a} = \frac{\partial \mathcal{L}}{\partial c} \times \frac{1}{b}∂a∂L​=∂c∂L​×b1​
  • ∂L∂b=∂L∂c×(−ab2)\frac{\partial \mathcal{L}}{\partial b} = \frac{\partial \mathcal{L}}{\partial c} \times (-\frac{a}{b^2})∂b∂L​=∂c∂L​×(−b2a​)

注意：除法对分母的梯度是负的，且当 bbb 接近0时梯度会爆炸。

7.5 激活函数

ReLU：f(x)=max⁡(0,x)f(x) = \max(0, x)f(x)=max(0,x)

前向：y=max⁡(0,x)y = \max(0, x)y=max(0,x)

反向：

  • dydx={1if x>00if x≤0\frac{dy}{dx} = \begin{cases} 1 & \text{if } x > 0 \\ 0 & \text{if } x \leq 0 \end{cases}dxdy​={10​if x>0if x≤0​

梯度传递：

  • ∂L∂x=∂L∂y×{1if x>00if x≤0\frac{\partial \mathcal{L}}{\partial x} = \frac{\partial \mathcal{L}}{\partial y} \times \begin{cases} 1 & \text{if } x > 0 \\ 0 & \text{if } x \leq 0 \end{cases}∂x∂L​=∂y∂L​×{10​if x>0if x≤0​

特点：正区间梯度=1，直接传递；负区间梯度=0，被截断。

Sigmoid：f(x)=11+e−xf(x) = \frac{1}{1+e^{-x}}f(x)=1+e−x1​

前向：y=11+e−xy = \frac{1}{1+e^{-x}}y=1+e−x1​

反向：

  • dydx=y(1−y)\frac{dy}{dx} = y(1-y)dxdy​=y(1−y)

梯度传递：

  • ∂L∂x=∂L∂y×y(1−y)\frac{\partial \mathcal{L}}{\partial x} = \frac{\partial \mathcal{L}}{\partial y} \times y(1-y)∂x∂L​=∂y∂L​×y(1−y)

特点：导数最大值为0.25，在饱和区（∣x∣|x|∣x∣ 很大时）梯度接近0。

Tanh：f(x)=tanh⁡(x)f(x) = \tanh(x)f(x)=tanh(x)

前向：y=tanh⁡(x)y = \tanh(x)y=tanh(x)

反向：

  • dydx=1−y2\frac{dy}{dx} = 1 - y^2dxdy​=1−y2

梯度传递：

  • ∂L∂x=∂L∂y×(1−y2)\frac{\partial \mathcal{L}}{\partial x} = \frac{\partial \mathcal{L}}{\partial y} \times (1 - y^2)∂x∂L​=∂y∂L​×(1−y2)

7.6 求和 c=∑iaic = \sum_i a_ic=∑i​ai​

前向：c=a1+a2+⋯+anc = a_1 + a_2 + \cdots + a_nc=a1​+a2​+⋯+an​

反向：

  • ∂c∂ai=1\frac{\partial c}{\partial a_i} = 1∂ai​∂c​=1（对每个 iii）

梯度传递：每个输入得到相同的梯度。

7.7 取负 c=−ac = -ac=−a

前向：c=−ac = -ac=−a

反向：

  • ∂c∂a=−1\frac{\partial c}{\partial a} = -1∂a∂c​=−1

梯度传递：∂L∂a=−∂L∂c\frac{\partial \mathcal{L}}{\partial a} = -\frac{\partial \mathcal{L}}{\partial c}∂a∂L​=−∂c∂L​

网络结构：

8.2 用一个预测不准的例子演示

设 y=20y = 20y=20（真实值），其他不变。

前向传播：

  • h1=5h_1 = 5h1​=5，h2=5h_2 = 5h2​=5

  1. ∂L∂y^=y^−y=10−20=−10\frac{\partial \mathcal{L}}{\partial \hat{y}} = \hat{y} - y = 10 - 20 = -10∂y^​∂L​=y^​−y=10−20=−10

  2. ∂y^∂w5=a1=5\frac{\partial \hat{y}}{\partial w_5} = a_1 = 5∂w5​∂y^​​=a1​=5，∂y^∂a1=w5=1\frac{\partial \hat{y}}{\partial a_1} = w_5 = 1∂a1​∂y^​​=w5​=1

    • ∂L∂w5=−10×5=−50\frac{\partial \mathcal{L}}{\partial w_5} = -10 \times 5 = -50∂w5​∂L​=−10×5=−50

9. 自动微分：让计算机帮你算梯度

上面手算梯度很麻烦。实际中我们用自动微分（Automatic Differentiation），让计算机自动帮我们算。

核心思想：每个变量都记录自己的值和梯度，在做运算时自动应用链式法则。

这个实现的关键是：

  1. 拓扑排序：确定计算顺序（从输出到输入）
  2. 局部梯度：每个运算定义自己的 _backward 函数
  3. 链式法则：在 _backward 中自动应用 ∂L∂x=∂L∂y⋅∂y∂x\frac{\partial \mathcal{L}}{\partial x} = \frac{\partial \mathcal{L}}{\partial y} \cdot \frac{\partial y}{\partial x}∂x∂L​=∂y∂L​⋅∂x∂y​

10. 总结

通过身高预测的例子，我们介绍了：

  1. 线性函数：最简单的模型 y=w1x1+w2x2y = w_1 x_1 + w_2 x_2y=w1​x1​+w2​x2​
  2. 激活函数：引入非线性 y=f(wx)y = f(wx)y=f(wx)
  3. 多层网络：组合多个层 y=f(W2f(W1x))y = f(W_2 f(W_1 x))y=f(W2​f(W1​x))
  4. 损失函数：衡量预测和真实的差距 L=12(y^−y)2\mathcal{L} = \frac{1}{2}(\hat{y} - y)^2L=21​(y^​−y)2
  5. 梯度下降：用梯度更新参数 w←w−η∂L∂ww \leftarrow w - \eta \frac{\partial \mathcal{L}}{\partial w}w←w−η∂w∂L​
  6. 反向传播：高效计算梯度的算法（链式法则）
  7. 各种运算的梯度：加、减、乘、除、激活函数等
  8. 自动微分：让计算机自动算梯度（如 micrograd）

这些是深度学习的基石。理解了这些，你就能理解 Transformer、大语言模型等更复杂的模型是如何训练的了。
